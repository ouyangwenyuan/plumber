Importing the project into Unity 3D

When Unity starts, click "Open" button and navigate to your unzipped folder Unblock The Ball Project

Project Structure

The main folder of the project is "Assets" folder. The "Assets" folder also contains subfolders:

  • Chartboost - contains Chartboost SDK
  • GoogleMobileAds - contains Admob SDK
  • Editor - contains scripts to use for built-in puzzle editor
  • EditorScripts - contains additional scripts for editor
  • Fonts - contains UI fonts
  • Materials - may contain some additional files for physics or particles
  • Plugins - contains android and ios related plugins and files
  • Prefabs - contains reusable game objects (Prefabs)
  • Scenes - contains all game scenes and levels
  • Scripts - contains all the game logic (C# scripts)
  • Sounds - contains sound assets
  • Sprites - contains all textures

 

Building a signed APK file

  • Select "File->Build Settings" from the main menu
  • If Android platform is not specified, select Android platform and click "Switch Platform" button
  • Click on "Player Settings" button
  • Change the "Product Name" option. 
  • Change the Bundle Identifier from "Other Settings" tab
  • Select your keystore file or create a new one from "Publishing Settings"
  • Click on "Build" or "Build & Run" button.

How to switch Ads Platform

Go to "File->Build Settings". Click on "Player Settings" button. Expand the Other Settings tab, find text field Scripting Define Symbols. 

  • If you are going to use Admob, type ADMOB there and press ENTER key.
  • If you are going to use Chartboost, type CHARTBOOST there and press ENTER key.

Admob

All the code regarding Ads is located in the AdsController.cs file from the "Assets/Scripts" folder. Here is the most important parts of the script:

  • private string AndroidBannerID = "ANDROID_BANNER_ID"; replace the bold text with your banner id
  • private string AndroidInterstitialID = "ANDROID_INTERSTITIALS_ID"; replace the bold text with your interstitials ad id
  • private string AndroidRewardedVideoID = "ANDROID_REWARDED_VIDEO_ID"; replace the bold text with your rewarded video ad id

 

  • public int adsCounter = 3; How often the interstitials (fullscreen) ads will be shown. That means after every 3rd time the game starts, the ads will be shown
  • The method void didCompleteRewardedVideo() Inside this method is a line GameManager.hintCount += 3;   which means amount of hints rewarded for watching a video
  • The method ShowRewardedVideo() is used for showing rewarded video ads
  • The method ShowInterstitialAds() is used for showing interstitials ads
  • The method ShowBanner() is used for showing the banner ads

Share Link and Rate Us button

Open the GameManager.cs in "Assets/Scripts" folder. Find the line

private string packageName="com.sandboxgames.unblocktheballslidingpuzzle";

and replace the package name string with your own package name.

How to change images and sounds

All the images are located in the "Assets/Sprites" folder. All the sounds are located in the "Assets/Sounds" folderFor changing an image or a sound, just replace it with your own.

Tile images must be combined into one file. This can be achieved in any graphic editor. Currently used tileset has 160x160 pixels per one tile + 2 pixels margin and padding between tiles.

How to use the Level Editor

Video tutorial: https://www.youtube.com/watch?v=8NbAtFRdAVs&t=3s

Here is a quick guide for creating a new level pack and editing puzzles:

  • All the levels for a level pack are in an appropriate folder. So in order to create a new level pack, first create a new folder inside "Scenes" folder. You can start building a new level by using TemplateScene file inside "Scenes" folder. Duplicate it and drag and drop into a newly created folder. Rename it according to the next important rules:  A level name must be started with "LP[pack number]_" and then "level-[level number]" should go. For example, if your newly created pack is 5th, so the level name must be started with LP5_ strictly. So the first level must be named LP5_level-1. Next level will be LP5_level-2 and so on.
  • Open the level scene. In the Hierarchy view, there is a Board game object, which also contains Tiles game object.
  • Once you click on Tiles game object, the editor will be activated.
  • Go to "Window->Tile Picker". The Tile Picker window should appear. Now you can choose any tile in the Tile Picker window. 
  • Note that the editor is only available if Tiles game object is selected from the Hierarchy view. So if you lost it, just select Tiles game object again.
  • For editing tiles, the Scene view must be focused on. Just click anywhere inside the Scene View to focus on it.

How to use the editor

  • Move the mouse cursor within the editor. The tile position will be calculated automatically
  • Press Shift key in order to add a new tile
  • Press Alt key in order to delete a tile
  • First of all, build a completed path. The completed path must always be started with Start Tile and must be always be ended with Goal Tile.
  • When the completed path is done, add some additional fake tiles for more challenge.
  • Add 3 stars to any tiles in the completed path: use S  key for adding or deleteng a star. The completed path must contain 3 stars.
  • When the path was done, click  "Generate Hint Path". 
  • For further editing, it is useful to temporarily hide the hint path. Use H key for hiding or showing the hint path.
  • Now you have to shuffle the tiles. Point to a tile with the mouse cursor and hold Control key and move the mouse cursor.
  • After you have created the completed path, the hint path, have added stars and have shuffled tiles, don't forget to enable the hint path and save the scene. Go to "File->Save Scene". 

Use the same way to create all other levels. Open GameMabager.cs file and edit maxPacks variable according to your count of level packs.
